# Workspace `src`

这个目录是 ROS 2 工作空间的 `colcon build` 入口。

真正的项目代码放在各个 `projects/*/src/` 里面。这里保留链接，方便你继续使用标准 ROS 2 命令：

```bash
cd ~/ros2_ws
colcon build
source install/setup.bash
```
