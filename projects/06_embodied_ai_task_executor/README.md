# Project 06: Embodied AI Task Executor

## Status

Planned.

## Goal

Build a high-level task executor that converts natural-language-style goals into robot actions.

## Planned ROS 2 Skills

- Task planning node
- Structured command parsing
- Navigation action integration
- Perception result integration
- Manipulation action integration
- System-level launch files
- Clear logs for debugging robot decisions

## Planned Deliverables

- A task executor node
- A small command language or natural-language-style interface
- Integration with navigation, perception, and manipulation demos
- End-to-end scenario demo
- README architecture diagram
