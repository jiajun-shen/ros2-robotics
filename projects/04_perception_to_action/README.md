# Project 04: Perception To Action

## Status

Planned.

## Goal

Build a perception pipeline where robot behavior changes based on camera or detection results.

## Planned ROS 2 Skills

- Camera topics
- OpenCV image processing
- Detection result topics
- Custom messages
- Behavior logic nodes
- Visualization markers

## Planned Deliverables

- Camera subscriber node
- Object or color detection node
- Detection publisher
- Behavior controller that reacts to detections
- README with sample outputs and diagrams

## Resume Bullet

```text
Built a perception-to-action ROS 2 pipeline using camera topics, OpenCV-based detection, custom messages, and behavior control nodes.
```

## Why It Matters

This project starts moving toward embodied intelligence: perception becomes an input to robot action.
